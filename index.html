User Manual
1
Create Sections

Navigate to Section Management to add class sections:

  • Enter the Program (e.g., BSCpE, BSIT, BSCS)
  • Select the Year Level (1st to 5th Year)
  • Enter the Section Name (e.g., A, B, C)
  • Click Add Section to save

Example: BSCpE 1st Year A → Creates "BSCpE 1st Year A"

2
Add Subjects

Go to Subject Management to add subjects:

  • Select a Section from the dropdown
  • Enter the Subject Code (e.g., CpE 311, IT 221)
  • Enter the Subject Name (e.g., Programming 1, Calculus)
  • Click Add Subject to save
  • Check "Apply to all sections of the same year level" to add a subject to multiple sections at once
3
Create Rooms

Go to Room Management to add rooms:

  • Enter the Room Name (e.g., Room 101, Lab A)
  • Enter the Capacity (e.g., 30, 50)
  • Click Add Room to save

The system prevents double-booking of rooms at the same time.

4
Assign Faculty Loads

In Faculty Loading, assign faculty to sections and subjects:

  • Enter the Faculty name (e.g., Dr. Cruz, Prof. Santos)
  • Select a Section from the dropdown
  • Select a Subject from the dropdown (filtered by section)
  • Choose the Day of the week
  • Set the Start Time and End Time
  • Select a Room (optional)
  • Enter Units (optional, 1-5)
  • Click Add Load or press Ctrl + Enter to save

The system automatically checks for scheduling conflicts.

5
Search, Filter & Sort

Use the search bar to find records quickly:

  • Type keywords to search across all fields
  • Select a Search Category (All, Faculty, Subject, Section)
  • Use the Day Filter to view loads by specific day
  • Click on sortable column headers (Faculty, Section, Subject) to sort A-Z or Z-A
6
Edit & Delete Records

Manage your schedule entries:

  • Click the Edit (pencil) button to modify an entry
  • Click the Delete (trash) button to remove an entry
  • Click any table row to populate the form with that record
  • Drag a table row onto a form field to auto-fill it
  • Use Reset Semester in the sidebar to clear all data (export first!)
7
Import & Export Data

Import or export your data:

  • Click Import to upload CSV or Excel files
  • Drag & drop a file onto the table area to import
  • Click Export to download all records as Excel (.xlsx)

Required import headers: Faculty, Section, Subject, Day, Start Time, End Time

8
View Faculty List

Go to Faculty List to see a summary of all faculty:

  • View each faculty member's total Loads and Units
  • See which Sections and Subjects they teach
  • Use the search box to filter faculty by name, section, or subject
  • The list is automatically populated from the Faculty Loading entries
9
View Section Schedules

View a visual schedule for any section:

  • Click any section button on the Dashboard
  • See a time grid showing classes from 7:00 AM to 11:00 PM
  • The 12:00–1:00 PM slot is marked as BREAK TIME
  • Click Export CSV to download the schedule
  • Each cell shows the faculty, subject, time, and room
10
Dashboard Overview

The Dashboard gives you a quick overview:

  • Loads — total number of schedule entries
  • Faculty — unique faculty members assigned
  • Section — unique sections created
  • Subject — unique subjects added
  • Use Quick Actions to navigate to any management view
11
Tips & Best Practices
  • Create sections first before adding subjects or faculty loads
  • Add all subjects to each section before creating faculty loads
  • Export regularly for backup before making major changes
  • Use Ctrl + Enter to quickly submit the form
  • After adding a load, Faculty and Section values are retained for quick re-entry
  • The system automatically checks for scheduling conflicts before saving
